# Portfolio Management Dashboard

Full-stack portfolio dashboard: React + Vite, Express + TypeScript, PostgreSQL + Prisma, JWT auth.

## Run with Docker

```bash
docker compose up --build
```

- Frontend: http://localhost:3000  
- API: http://localhost:5000/api
